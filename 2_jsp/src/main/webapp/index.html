(2) 지시어

JSP 페이지에서 전체에 영향을 미치는 설정(정보)을 작성할 때 사용
<%@ page|includ|taglib 속성="속성값" 속성="속성값" %>

  1. page 지시어 : 현재 jsp 페이지를 처리하는데 필요한 속성들을 작성하는 부분이다
    • language : 사용할 언어유형 지정
    • contentType : 해당 페이지의 형식, 문자셋 지정
    • pageEncoding : 해당 페이지 내에 기록된 자바코드의 인코딩 방식
    • import : 자바의 import와 같은 의미
    • errorPage : 현재 페이지에서 오류가 발생했을 경우 보여줄 에러 페이지 경로 지정
    view datail
  2. include 지시어 : 현재 JSP 페이지 내에 또 다른 jsp 페이지를 포함시키고자 할 때 사용하는 지시어 include
  3. taglib 지시어 : JSP 기능을 확장시킬 수 있는 추가적인 라이브러리를 등록할 때 사용
